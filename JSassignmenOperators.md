# JavaScript Assignment Operators

Assignment operators assign values to JavaScript variables.

| Operator | Example | Same As
| --- | --- | --- |
| = | x = y | x = y |
| += | x += y | x = x + y |
| -= | x -= y |	x = x - y |
| *= | x *= y | x = x * y |
| /= | x /= y | x = x / y |
| %= | x %= y |	x = x % y |
| **= | x **= y | x = x ** y |

[Additional details](https://www.w3schools.com/js/js_assignment.asp)
